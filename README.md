# creation

创作类工作，包括：UI设计、视频编辑、文章创作等

## 分类

### 图

* icon 图标
* logo 标志
* gif 动图
* social 社交媒体图片（微信等）
* prototype 原型设计
* banner 网站海报、新闻插图等
* print 海报宣传页信函卡票邀请函
* infographic 信息图

### 多媒体

* sound 音效（简短的声音，例如：鸟叫）
* audio 配音（场景的长声，例如：一段激昂的进行曲）
* video 视频

### 文章

* legal 法律文书（协议合同等）
* article （文章）
